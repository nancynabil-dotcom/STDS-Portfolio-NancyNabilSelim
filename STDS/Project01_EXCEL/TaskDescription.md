## Project Overview
This project is an interactive **Excel Dashboard** designed to analyze and visualize e-commerce business data.
It helps identify trends in **sales**, **profit**, and **customer behavior** across different **regions**, **product categories**, and **customer segments**.

---

## Data Preprocessing
All data preprocessing steps were performed directly in **Excel using Power Query**, including:
- Removing duplicate records and handling missing values
- Cleaning and standardizing categorical fields
- Converting data types to ensure consistency
- Creating calculated columns such as **Profit** and **Profit Margin**
- Preparing clean and structured data suitable for dashboard creation

---

## Data Analysis
The cleaned data was analyzed using:
- Pivot Tables to summarize key metrics
- Aggregations for sales, profit, and quantity
- Grouping data by region, category, and customer segment

---

## Dashboard Features
The Excel dashboard includes the following components:
- **KPI Cards:** Total Sales, Total Profit, Quantity Sold, and Profit Margin
- **Charts:**  
  - Sales and profit performance by region  
  - Product category analysis  
  - Customer segment comparison
- **Slicers:** Region, Customer Segment, and Product Category for interactive filtering
- **Navigation Buttons:** Used to enable smooth transitions between dashboard sections
- A clean and user-friendly layout for easy data exploration
